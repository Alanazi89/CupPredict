# CupPredict

Production-ready Next.js 15 App Router starter for explainable World Cup predictions.

## Stack

- Next.js 15, React 19, TypeScript
- Tailwind CSS v4 and shadcn/ui conventions
- TanStack Query, React Hook Form, Zod, Recharts, Framer Motion
- ESLint, Prettier, Husky, lint-staged

## Scripts

```bash
npm install
npm run dev
npm run typecheck
npm run build
```

Copy `.env.example` to `.env.local` to override public metadata settings.
